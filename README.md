# 🧠 Totem Inteligente em Libras

## 📌 Descrição

Este projeto consiste no desenvolvimento de um protótipo de totem inteligente para atendimento inclusivo no setor público, utilizando tecnologias de visão computacional e inteligência artificial para reconhecimento de sinais da Língua Brasileira de Sinais (Libras).

O sistema tem como objetivo facilitar a comunicação entre pessoas surdas e atendentes, promovendo acessibilidade e inclusão digital.

---

## 🚀 Funcionalidades

* Captura de vídeo em tempo real
* Estrutura modular do sistema
* Sistema de logs para monitoramento
* Base preparada para reconhecimento de gestos (em desenvolvimento)

---

## 🛠️ Tecnologias utilizadas

* Python
* OpenCV
* MediaPipe (em integração)
* Flask (backend)
* PostgreSQL (armazenamento de dados)
* gTTS (conversão texto-voz)
